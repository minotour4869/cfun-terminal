# CodeFun-Terminal

Codefun Terminal is a small project to parse, submit page [codefun.vn](https://codefun.vn/) problems and get user infomation.
Project is made by Python. UwU

Currently project is in beta version, and only tested on Windows. If you occur any problem about my project (bug, feature...), feel free to ask me. You are welcome.

## INSTALLATION

1. Install Python 3.
2. Clone this Repository or Download as ZIP file and extract it.
3. Install Chrome driver and add it in your system variables.
4. (Optimal) Add `cfun.exe` (when it released) path to your system variables (for easy interaction, i guess?)
5. Run the following command (when you installed Python and PIP configured properly)
`py -m pip install -r requirements.txt`
